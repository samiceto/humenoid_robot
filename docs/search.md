# Search Functionality

The Physical AI & Humanoid Robotics book includes powerful search functionality powered by Algolia, allowing you to quickly find information across all 18 chapters and supporting materials.

## Using the Search

The search bar is available in the top navigation bar of every page. Simply click on the search icon or press `Ctrl + K` (or `Cmd + K` on Mac) to open the search interface.

## Search Tips

- Type keywords related to robotics, AI, ROS 2, Isaac Sim, control systems, etc.
- Use quotes for exact phrase matching: `"whole body control"`
- The search covers all book content, code examples, and technical documentation
- Results are ranked by relevance to your query

## Search Coverage

The search index includes:
- All 18 chapters of the textbook
- Code examples and technical explanations
- Technical terminology and concepts
- ROS 2 commands and packages
- Isaac Sim and Isaac ROS components
